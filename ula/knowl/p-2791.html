  1. Form the matrix

    \begin{equation*} P_{\bcal} = \left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ 1 \amp 1 \amp 0 \amp 0 \\ 1 \amp 1 \amp 1 \amp 0 \\ 1 \amp 1 \amp 1 \amp 1 \end{array}\right] \sim \left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \end{array}\right]\text{,} \end{equation*}

    which shows that the vectors are linearly independent and span \(\real^4\text{.}\)

  2. We have \(\xvec = P_{\bcal}\coords{\xvec}{\bcal}\text{.}\)

  3. We have \(\coords{\xvec}{\bcal} = P_{\bcal}^{-1}\xvec\) where

    \begin{equation*} P_{\bcal}^{-1} = \left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ -1 \amp 1 \amp 0 \amp 0 \\ 0 \amp -1 \amp 1 \amp 0 \\ 0 \amp 0 \amp -1 \amp 1 \end{array}\right]\text{.} \end{equation*}

  4. We find \(\coords{\xvec}{\bcal}=\fourvec{23}{-11}{-2}{9}\text{.}\)

  5. We find \(\xvec=\fourvec{3}{4}{1}{-3}\text{.}\)
