  1. If we write \(\xvec_k=\threevec{J_k}{Y_k}{A_k}\text{,}\) we have

    \begin{equation*} \xvec_{k+1} = \left[\begin{array}{rrr} 0 \amp 0 \amp 0.5 \\ 0.9 \amp 0 \amp 0 \\ 0.6 \amp 0.8 \amp 0 \\ \end{array}\right] \xvec_k\text{.} \end{equation*}

  2. We find the eigenvalues \(\lambda_1 = 0.85\) and \(\lambda_{2,3} = -0.43\pm 0.49i\text{.}\)

  3. All three eigenvalues have a length less than one. Therefore, the populations will eventually become extinct.

  4. With a birth rate of \(80\)%, the first eigenvalue goes up to \(\lambda_1=1.02\text{,}\) which means that, eventually, all the populations grow by \(2\)%. An eigenvector is approximately \(\vvec_1 = \threevec{78}{69}{100}\) so there are about \(78\) juveniles and \(69\) yearlings for every \(100\) adults.
