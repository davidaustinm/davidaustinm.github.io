  3. We see that \(A \sim \left[\begin{array}{rrrr} {\mathbf 1} \amp 0 \amp 0 \amp 2 \\ 0 \amp {\mathbf 1} \amp 0 \amp -1 \\ 0 \amp 0 \amp 1 \amp {\mathbf 1} \\ \end{array}\right]\text{.}\)

  5. For any \(\bvec\text{,}\) the augmented matrix \(\left[\begin{array}{r|r} A \amp \bvec \end{array}\right]\) will have exactly one column without a pivot. Therefore, the solution space will form a line in \(\real^4\text{,}\) as in part b.
