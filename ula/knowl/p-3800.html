  1. The characteristic polynomial is \(\det(A-\lambda I) = (2-\lambda)^2\text{.}\) There is a single eigenvalue \(\lambda=2\) having multiplicity \(2\text{.}\)

  2. The characteristic polynomial is \(\det(A-\lambda I) = (3-\lambda)(4-\lambda)(-6-\lambda)\text{.}\) There are three eigenvalues \(\lambda=3, 4, -6\text{,}\) each of multiplicity \(1\text{.}\)

  3. The characteristic polynomial is \(\det(A-\lambda I) = (-2-\lambda)^2\text{,}\) showing that there is one eigenvalue \(\lambda=-2\) having multiplicity \(2\text{.}\)

  4. The characteristic polynomial is \(\det(A-\lambda I) = (3-\lambda)(-2-\lambda)\text{.}\) There are two eigenvalues \(\lambda=3\) and \(\lambda=-2\text{,}\) each having multiplicity \(1\text{.}\)
