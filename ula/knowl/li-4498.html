Item 6.4.4.5.b.

The vector \(\vvec_3=2\vvec_1-\vvec_2\) so this set of vectors is linearly dependent. Since \(\vvec_3\) is in the span of \(\vvec_1\) and \(\vvec_2\text{,}\) projecting \(\vvec_3\) into that subspace gives \(\widehat{\vvec}_3 = \vvec_3\) so that \(\wvec_3 = \vvec_3 - \widehat{\vvec}_3 = \zerovec\text{.}\)
