Solution 5.1.4.10.1.

  1. We see that

    \begin{equation*} \begin{array}{c} P=\left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \\ \end{array}\right], L=\left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ -1 \amp 1 \amp 0 \amp 0 \\ 0 \amp -1 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \amp 1 \\ \end{array}\right], \\ U=\left[\begin{array}{rrrr} -1 \amp 1 \amp 0 \amp 0 \\ 0 \amp -1 \amp 1 \amp 0 \\ 0 \amp 0 \amp -1 \amp 1 \\ 0 \amp 0 \amp 0 \amp 0 \\ \end{array}\right]\text{.} \end{array} \end{equation*}

  2. Because \(\nul(A) = \nul(U)\text{,}\) we see that \(\dim~\nul(A)=1\) with a basis vector \(\fourvec1111\text{.}\)

  3. It is not true that \(\col(A) = \col(L)\text{.}\) Since there is a nonzero vector \(\xvec\) such that \(A\xvec=\zerovec\text{,}\) we know that \(A\) is not invertible and hence its column space \(\col(A)\) is not \(\real^4\text{.}\) However, \(L\) is invertible so we have \(\col(L)=\real^4\text{.}\)
