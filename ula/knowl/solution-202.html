Solution 4.3.2.1.

  1. We have \(A\xvec=2\lambda_1\vvec_1-3\lambda_2\vvec_2= 6\vvec_1+3\vvec_2\text{.}\)

  2. By the results of the previous part of this activity, we have \(\coords{A\xvec}{\bcal} = \twovec63\text{.}\)

  3. If \(\coords{\xvec}{\bcal} = \twovec{c_1}{c_2}\text{,}\) then \(\xvec=c_1\vvec_1+c_2\vvec_2\text{.}\) This means that \(A\xvec = c_1\lambda_1\vvec_1 + c_2\lambda_2\vvec_2\) so that \(\coords{A\xvec}{\bcal} = \twovec{\lambda_1c_1}{\lambda_2c_2}= \twovec{3c_1}{-c_2}\text{.}\)

  4. We notice that \(D\twovec{c_1}{c_2} = \twovec{3c_1}{-c_2}\text{,}\) which tells us that \(\coords{A\xvec}{\bcal} = D\coords{\xvec}{\bcal}\text{.}\)

  5. Remember that \(\coords{\yvec}{\bcal} = C_{\bcal}^{-1}\yvec\) for any vector \(\yvec\text{.}\) Since \(\coords{A\xvec}{\bcal} = D\coords{\xvec}{\bcal}\text{,}\) we know that \(C_{\bcal}^{-1}A\xvec = DC_{\bcal}^{-1}\xvec\) and hence \(C_{\bcal}^{-1}A = DC_{\bcal}^{-1}\text{.}\)

  6. Multiplying both sides of this relationship by \(C_{\bcal}\text{,}\) we obtain \(A = C_{\bcal}DC_{\bcal}^{-1}\text{,}\) which can be verified using Sage.
