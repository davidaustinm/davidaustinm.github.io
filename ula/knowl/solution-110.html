Solution 3.1.1.1.

  1. We can view the process of dividing by \(3\) as, instead, multiplying by the multiplicative inverse \(\frac13\) of \(3\text{.}\) We have

    \begin{equation*} \begin{array}{rl} \frac13(3x) \amp = \dfrac13 5 \\ \left(\frac13 3\right)x\amp = \dfrac53 \\ 1x \amp = \dfrac 53 \\ x \amp = \dfrac53\text{.} \\ \end{array} \end{equation*}

  2. As we have seen a few times, the matrix is \(A=\left[\begin{array}{rr} 0 \amp -1 \\ 1 \amp 0 \\ \end{array}\right]\text{.}\)

  3. Here, the matrix is \(B=\left[\begin{array}{rr} 0 \amp 1 \\ -1 \amp 0 \\ \end{array}\right]\text{.}\)

  4. We should expect that \(BA=I\) since the effect of rotating by \(90^\circ\) counterclockwise followed by rotating \(90^\circ\) clockwise is to leave a vector unchanged. We can verify this by performing the matrix multiplication.

  5. We have

    \begin{equation*} \left[\begin{array}{rr|r} 0 \amp -1 \amp 3 \\ 1 \amp 0 \amp -2 \\ \end{array}\right] \sim \left[\begin{array}{rr|r} 1 \amp 0 \amp -2 \\ 0 \amp 1 \amp -3 \\ \end{array}\right] \end{equation*}

    so the solution is \(\xvec=\twovec{-2}{-3}\text{.}\)

  6. As in the first part of the problem, we have

    \begin{equation*} \begin{array}{rl} A\xvec \amp = \twovec{3}{-2} \\ B(A\xvec) \amp = B\twovec{3}{-2} \\ (BA)\xvec \amp = B\twovec{3}{-2} \\ I\xvec \amp = B\twovec{3}{-2} \\ \xvec \amp = B\twovec{3}{-2}=\twovec{-2}{-3}\text{.} \\ \end{array} \end{equation*}
