Item 6.2.5.6.b.

  1. If \(D\) is diagonal, then \(D^T = D\text{.}\)

  2. \((BAB^{-1})^T = (B^{-1})^TA^TB^T\) so this matrix need not be symmetric.

  3. \((AA^T)^T = (A^T)^TA^T = AA^T\) so this matrix is symmetric.

  4. \((BDB^T)^T = (B^T)^TD^TB^T = BDB^T\) so this matrix is symmetric.
