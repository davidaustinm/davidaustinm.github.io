Item 2.2.6.10.a.

Since the solution is unique, there must be a pivot position in every column of \(A\text{.}\) This means that there are four pivot positions so there is a pivot position in every row. In other words, \(A \sim \left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \\ \end{array}\right] \text{.}\)
