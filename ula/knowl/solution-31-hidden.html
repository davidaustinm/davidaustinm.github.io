1. We find the linear system

    \begin{equation*} \begin{alignedat}{4} 4c_1 \amp {}+{} \amp c_2 \amp {}-{} \amp 2c_3 \amp {}={} \amp 0 \\ \amp \amp -3c_2 \amp {}+{} \amp c_3 \amp {}={} \amp 1 \\ 2c_1 \amp {}+{} \amp 3c_2 \amp {}+{} \amp c_3 \amp {}={} \amp 2 \\ c_1 \amp {}+{} \amp c_2 \amp \amp \amp {}={} \amp -2 \\ \end{alignedat} \end{equation*}

    with corresponding augmented matrix

    \begin{equation*} \left[\begin{array}{rrr|r} 4 \amp 1 \amp -2 \amp 0 \\ 0 \amp -3 \amp 1 \amp 1 \\ 2 \amp 3 \amp 1 \amp 2 \\ 1 \amp 1 \amp 0 \amp -2 \end{array}\right] \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \end{array}\right]\text{.} \end{equation*}

    This shows that the linear system is inconsistent so there are no such weights \(c_1\text{,}\) \(c_2\text{,}\) and \(c_3\text{.}\) This means that \(\bvec\) is not a linear combination of \(\vvec_1\text{,}\) \(\vvec_2\text{,}\) and \(\vvec_3\text{.}\)

  2. We find vectors

    \begin{equation*} \vvec_1=\threevec{3}{1}{-1}, \vvec_2=\threevec{2}{0}{-1}, \vvec_3=\threevec{-1}{2}{3}, \bvec=\threevec{4}{0}{1}\text{.} \end{equation*}

  3. This is the same as asking if the linear system corresponding to the following augmented matrix is consistent:

    \begin{equation*} \left[\begin{array}{rrr|r} 0 \amp 1 \amp 2 \amp -1 \\ -2 \amp 1 \amp 0 \amp 3 \\ 1 \amp -1 \amp -1 \amp -1 \end{array}\right] \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp 1 \amp -2 \\ 0 \amp 1 \amp 2 \amp -1 \\ 0 \amp 0 \amp 0 \amp 0 \end{array}\right]\text{.} \end{equation*}

    From the reduced row echelon form, we see that the system is consistent, which means that \(\bvec\) can be expressed as a linear combination of \(\vvec_1\text{,}\) \(\vvec_2\text{,}\) and \(\vvec_3\text{.}\) Moreover, there are infinitely ways in which to do so.

  4. No, it is not possible to write every three-dimensional \(\bvec\) as a linear combination of \(\vvec_1\text{,}\) \(\vvec_2\text{,}\) and \(\vvec_3\) because the matrix \(\left[\begin{array}{rrr} \vvec_1 \amp \vvec_2 \amp \vvec_3 \end{array}\right]\) does not have a pivot position in every row. This means that, with some choice of vector \(\bvec\text{,}\) we will obtain an inconsistent system.

  5. We find the augmented matrix

    \begin{equation*} \left[\begin{array}{rrr|r} 0 \amp 1 \amp 1 \amp 0 \\ -2 \amp 1 \amp -1 \amp 8 \\ 1 \amp -1 \amp -2 \amp -4 \end{array}\right] \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp -4 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \end{array}\right]\text{.} \end{equation*}

    This shows that \(\bvec\) can be expressed as a linear combination of \(\vvec_1\text{,}\) \(\vvec_2\text{,}\) and \(\vvec_3\) in exactly one way.

  6. Every vector \(\bvec\) can be expressed as a linear combination of \(\vvec_1\text{,}\) \(\vvec_2\text{,}\) and \(\vvec_3\) in exactly one way because \(\left[\begin{array}{rrr} \vvec_1 \amp \vvec_2 \amp \vvec_3 \end{array}\right]\) has a pivot position in every row and every column.
