We find the linear system

\begin{equation*} \begin{alignedat}{4} 4c_1 \amp {}+{} \amp c_2 \amp {}-{} \amp 2c_3 \amp {}={} \amp 0 \\ \amp \amp -3c_2 \amp {}+{} \amp c_3 \amp {}={} \amp 1 \\ 2c_1 \amp {}+{} \amp 3c_2 \amp {}+{} \amp c_3 \amp {}={} \amp 2 \\ c_1 \amp {}+{} \amp c_2 \amp \amp \amp {}={} \amp -2 \\ \end{alignedat} \end{equation*}

with corresponding augmented matrix

\begin{equation*} \left[\begin{array}{rrr|r} 4 \amp 1 \amp -2 \amp 0 \\ 0 \amp -3 \amp 1 \amp 1 \\ 2 \amp 3 \amp 1 \amp 2 \\ 1 \amp 1 \amp 0 \amp -2 \end{array}\right] \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \end{array}\right]\text{.} \end{equation*}

This shows that the linear system is inconsistent so there are no such weights \(c_1\text{,}\) \(c_2\text{,}\) and \(c_3\text{.}\) This means that \(\bvec\) is not a linear combination of \(\vvec_1\text{,}\) \(\vvec_2\text{,}\) and \(\vvec_3\text{.}\)
