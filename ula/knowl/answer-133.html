Answer 3.4.5.9.1.

  1. \(\det(A) = 2x-y-2z = 0\text{.}\)

  2. If we replace the third column by either \(\vvec_1\) or \(\vvec_2\text{,}\) we obtain a matrix whose columns are linearly dependent.

  3. If \(\det(A)=0\) for a vector \(\threevec xyz\text{,}\) then \(\threevec xyz\) is a linear combination of \(\vvec_1\) and \(\vvec_2\text{.}\)
