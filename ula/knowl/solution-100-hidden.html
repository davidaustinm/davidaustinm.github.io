We use the fact that the columns of the requested matrices have the form \(\left[\begin{array}{rr} T(\evec_1) \amp T(\evec_2) \end{array}\right]\text{.}\)

  1. \(\left[\begin{array}{rr} 1 \amp 0 \\ 0 \amp 1 \\ \end{array}\right] \text{.}\)

  2. \(\left[\begin{array}{rr} 0 \amp 1 \\ 1 \amp 0 \\ \end{array}\right] \text{.}\)

  3. The composition of this reflection with itself is described by multiplying the matrix by itself. This produces the matrix \(\left[\begin{array}{rr} 1 \amp 0 \\ 0 \amp 1 \\ \end{array}\right] \text{,}\) which we just saw is the matrix for the identity transformation. This means that reflecting a vector in the line \(y=x\) twice produces the original vector.

  4. \(\left[\begin{array}{rr} 0 \amp -1 \\ 1 \amp 0 \\ \end{array}\right] \text{.}\)

  5. If we first rotate and then reflect, we obtain the matrix transformation defined by

    \begin{equation*} \left[\begin{array}{rr} 0 \amp 1 \\ 1 \amp 0 \\ \end{array}\right] \left[\begin{array}{rr} 0 \amp -1 \\ 1 \amp 0 \\ \end{array}\right] = \left[\begin{array}{rr} 1 \amp 0 \\ 0 \amp -1 \\ \end{array}\right]\text{,} \end{equation*}

    which is the matrix for reflecting in the horizontal axis.

    If we first reflect and then rotate, we obtain the matrix

    \begin{equation*} \left[\begin{array}{rr} 0 \amp -1 \\ 1 \amp 0 \\ \end{array}\right] \left[\begin{array}{rr} 0 \amp 1 \\ 1 \amp 0 \\ \end{array}\right] = \left[\begin{array}{rr} -1 \amp 0 \\ 0 \amp 1 \\ \end{array}\right]\text{,} \end{equation*}

    which is the matrix for reflecting in the vertical axis.

  6. \(\left[\begin{array}{rr} 0 \amp -1 \\ 1 \amp 0 \\ \end{array}\right] \left[\begin{array}{rr} 0 \amp -1 \\ 1 \amp 0 \\ \end{array}\right] = \left[\begin{array}{rr} -1 \amp 0 \\ 0 \amp -1 \\ \end{array}\right] \text{,}\) which represents a \(180^\circ\) rotation.

  7. Composing four times corresponds to raising the matrix to the fourth power, which gives us the identity matrix \(I\text{.}\)

  8. If we consider the effect of rotating the vector \(\twovec{1}{0}\) by an angle \(\theta\text{,}\) we obtain the vector \(\twovec{\cos\theta}{\sin\theta}\text{.}\)
