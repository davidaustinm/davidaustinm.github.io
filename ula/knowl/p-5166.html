Notice that \(U\) is upper triangular, which means that it is straightforward to find its reduced row echelon form:

\begin{equation*} \begin{array}{rcl} U= \left[\begin{array}{rrrr} 3 \amp 2 \amp -1 \amp 0 \\ 0 \amp \frac{7}{3} \amp \frac{4}{3} \amp -1 \\ 0 \amp 0 \amp \frac{9}{7} \amp \frac{9}{7} \end{array}\right] \amp \sim \amp \left[\begin{array}{rrrr} 3 \amp 2 \amp -1 \amp 0 \\ 0 \amp 7 \amp 4 \amp -3 \\ 0 \amp 0 \amp 1 \amp 1 \end{array}\right] \\ \amp \sim \amp \left[\begin{array}{rrrr} 3 \amp 2 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \amp -1 \\ 0 \amp 0 \amp 1 \amp 1 \end{array}\right] \\ \amp \sim \amp \left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \amp -1 \\ 0 \amp 0 \amp 1 \amp 1 \end{array}\right]\text{.} \\ \end{array} \end{equation*}

This shows that a basis for \(\nul(C)=\nul(U)\) is \(\fourvec{-1}1{-1}1\text{.}\)
