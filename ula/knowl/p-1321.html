The equation \(\left[\begin{array}{rr} \vvec_1 \amp \vvec_2 \end{array}\right] \xvec = \bvec\) is consistent so \(\bvec\) is in \(\span{\vvec_1,\vvec_2}\text{.}\) Since \(\left[\begin{array}{rr} \vvec_1 \amp \vvec_2 \end{array}\right]\) has a pivot position in every row, \(\span{\vvec_1,\vvec_2}\) is \(\real^2\text{.}\)

  2. Let's consider the following vectors.

    1. The equation \(\left[\begin{array}{rrr}\vvec_1 \amp \vvec_2 \amp \vvec_3 \end{array}\right]\xvec = \bvec\) is consistent so \(\bvec\) is in \(\span{\vvec_1,\vvec_2,\vvec_3}\text{.}\)

    2. The vector \(\vvec_3\) is in \(\span{\vvec_1,\vvec_2}\) because \(\vvec_3 = \vvec_1-2\vvec_2\text{.}\)

    3. The equation \(\left[\begin{array}{rrr}\vvec_1 \amp \vvec_2 \amp \vvec_3 \end{array}\right]\xvec = \bvec\) is not consistent so \(\bvec\) is not in \(\span{\vvec_1,\vvec_2,\vvec_3}\text{.}\)

    4. \(\span{\vvec_1,\vvec_2,\vvec_3}\) is the plane defined by \(\vvec_1\) and \(\vvec_2\text{.}\)
