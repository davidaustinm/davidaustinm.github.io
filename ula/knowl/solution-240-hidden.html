1. We have \(\xvec_{k+1} = A\xvec_k\) where \(A = \left[\begin{array}{rrr} 0 \amp 0 \amp 8 \\ \frac 12 \amp 0 \amp 0 \\ 0 \amp \frac 14 \amp 0 \\ \end{array}\right]\text{.}\)

  2. We can use Sage to compute that \(A^3=I\text{.}\)

  3. The only eigenvalues of \(I\) are \(1\) so any eigenvalue \(\lambda\) of \(A\) must satisfy \(\lambda^3=1\text{.}\)

  4. Sage tells us that the eigenvalues are \(\lambda_1=1\) and \(\lambda_{2,3} = -\frac12\pm\frac{\sqrt3}{2}i\text{.}\)

  5. The eigenvector \(\vvec_1=\threevec841\) satisfies \(A\vvec_1 = \vvec_1\) so it is unchanged. The complex eigenvalues cause a \(120^\circ\) rotation about the axis defined by \(\vvec_1\text{.}\) Therefore, the trajectories lie on circles about this line.

  6. After every three years, the populations return to their initial values.

  7. If \(\xvec_0=\vvec_1 = \threevec841\text{,}\) we know that \(A\xvec_0 = \xvec_0\) so this population vector is unchanged.
