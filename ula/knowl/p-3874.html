  1. A vector lying along the line of reflection is unchanged so \(A\twovec{1}{1} = \twovec{1}{1}\text{,}\) which shows that \(\twovec{1}{1}\) is an eigenvector with associated eigenvalue \(\lambda = 1\text{.}\) At the same time, \(A\twovec{1}{-1} = -\twovec{1}{-1}\) so \(\twovec{1}{-1}\) is an eigenvector with associated eigenvalue \(\lambda=-1\text{.}\)

  2. Every vector \(\vvec\) satisfies \(A\vvec=-\vvec\) so every two-dimensional vector is an eigenvector with associated eigenvalue \(\lambda=-1\text{.}\)

  3. Vectors along the \(y\)-axis are unchanged so \(\threevec{0}{1}{0}\) is an eigenvector with associated eigenvalue \(\lambda=1\text{.}\) Vectors in the \(xz\)-plane are multiplied by \(-1\) so \(\threevec{1}{0}{0}\) and \(\threevec{0}{0}{1}\) are eigenvectors with associated eigenvalue \(\lambda=-1\text{.}\)

  4. The vector \(\threevec{1}{0}{0}\) is an eigenvector with associated eigenvalue \(\lambda=1\text{.}\) There are no other eigenvectors that are not scalar multiples of this one.
