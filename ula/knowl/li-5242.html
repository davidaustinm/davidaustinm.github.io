Item 7.2.4.7.a.

The matrix is symmetric because \((B^TB)^T = B^T(B^T)^T = B^TB\text{.}\) If \(\uvec\) is an eigenvector having unit length, then \(q(\uvec) = \uvec\cdot(B^TB\uvec) = \lambda\) and \(q(\uvec) = \uvec\cdot(B^TB\uvec) = (B\uvec)\cdot(B\uvec) = |B\uvec|^2 = \lambda \geq 0\text{.}\)
