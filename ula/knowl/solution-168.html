Solution 3.5.5.4.1.

  1. If \(BA\xvec=\zerovec\text{,}\) then \(A\xvec=\zerovec\) if \(B\) is invertible.

  2. The column space consists of all the vectors \(\bvec\) for which \(A\xvec=\bvec\) is consistent. We can rewrite this condition as \(AB(B^{-1}\xvec)=\bvec\text{,}\) which means that \(A\xvec=\bvec\) is consistent exactly when \(AB\xvec=\bvec\) is.

  3. If \(A\sim A'\text{,}\) then \(A\) and \(A'\) have the same reduced row echelon form, which determines \(\nul(A)\text{.}\)
