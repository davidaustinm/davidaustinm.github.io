We can, of course, repeat this process. The vector \(\xvec_1\) describes the state after one day. After a second day, we have the state vector

\begin{equation*} \xvec_2 = T(\xvec_1) = A\xvec_1 = A^2\xvec_0 = \ctwovec{600}{400}\text{.} \end{equation*}

We can continue this process finding \(\xvec_k\text{,}\) the state after \(k\) days using \(\xvec_k = A\xvec_{k-1} = A^k\xvec_0\text{.}\) In this way, we see that the long-term behavior of the state vector is determined by the powers of the matrix \(A\text{.}\)
