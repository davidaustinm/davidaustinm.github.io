Let's discuss a few of the issues that appear in this activity. First, if \(A\) is an \(m\times n\) matrix, we can form the matrix product \(A\xvec\) when \(\xvec\) is an \(n\)-dimensional vector in \(\real^n\text{.}\) The resulting product \(A\xvec\) is an \(m\)-dimensional vector in \(\real^m\text{.}\) If \(T(\xvec) = A\xvec\text{,}\) we therefore write \(T:\real^n\to\real^m\) meaning \(T\) takes vectors in \(\real^n\) as inputs and produces vectors in \(\real^m\) as outputs. For instance, if

\begin{equation*} A=\left[\begin{array}{rrrr} 4 \amp 0 \amp -3 \amp 2 \\ 0 \amp 1 \amp 3 \amp 1 \\ \end{array}\right]\text{,} \end{equation*}

then \(T:\real^4\to\real^2\text{.}\)
