No, it is not possible to write every three-dimensional \(\bvec\) as a linear combination of \(\vvec_1\text{,}\) \(\vvec_2\text{,}\) and \(\vvec_3\) because the matrix \(\left[\begin{array}{rrr} \vvec_1 \amp \vvec_2 \amp \vvec_3 \end{array}\right]\) does not have a pivot position in every row. This means that, with some choice of vector \(\bvec\text{,}\) we will obtain an inconsistent system.
