If we choose any vector \(\yvec\text{,}\) we have \(B(AB-I)\yvec = B[(AB-I)\yvec]=\zerovec\text{.}\) Therefore, \((AB-I)\yvec\) is a solution to the homogeneous equation \(B\xvec=\zerovec\text{,}\) and we can conclude that \((AB-I)\yvec = \zerovec\text{.}\) This means that \(AB\yvec - \yvec = \zerovec\) or \(AB\yvec = \yvec\) for every vector \(\yvec\text{.}\) Therefore, \(AB=I\text{,}\) which shows that \(B^{-1} = A\text{.}\)
