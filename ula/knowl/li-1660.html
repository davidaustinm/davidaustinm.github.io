Item b.

We note that

\begin{equation*} \left[\begin{array}{rrr} 1 \amp 0 \amp 1 \\ 1 \amp 1 \amp 0 \\ 1 \amp -1 \amp -1 \end{array}\right] \sim \left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{array}\right]\text{.} \end{equation*}

Since there is a pivot position in every row, the vectors span \(\real^3\text{.}\) Since there is a pivot position in every column, the vectors are linearly independent. Consequently, this set of vectors forms a basis for \(\real^3\text{.}\)
