1. Consider the augmented matrices

    \begin{equation*} \begin{array}{ccc} \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ \end{array}\right] \amp \left[\begin{array}{rrr|r} 1 \amp -2 \amp 0 \amp 2 \\ 0 \amp 0 \amp 1 \amp -3 \\ 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ \end{array}\right] \amp \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \amp -2 \\ 0 \amp 0 \amp 1 \amp 3 \\ 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ \end{array}\right]\text{.} \end{array} \end{equation*}

    These examples show that anything is possible: the system may or may not be consistent, and, if it is consistent, there may or may not be a unique solution.

  2. Consider the augmented matrix

    \begin{equation*} \left[\begin{array}{rrrrrr|r} 1 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \amp 0 \amp 0 \amp 0 \amp -1 \\ 0 \amp 0 \amp 1 \amp 0 \amp 0 \amp 0 \amp 4 \\ \end{array}\right] \end{equation*}

    There must be a column without a pivot position so we can guarantee that there are infinitely many solutions.

  3. Consider the augmented matrix

    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 3 \\ 0 \amp 1 \amp 0 \amp -2 \\ 0 \amp 0 \amp 1 \amp 1 \\ 0 \amp 0 \amp 0 \amp 0 \\ \end{array}\right]\text{.} \end{equation*}

    We know that every column of the coefficient matrix has a pivot position and that the rightmost column does not. There must be at least as many equations as variables.
