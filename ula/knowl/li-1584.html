Item 3.1.5.5.c.

We use a scaling followed by replacement operations:

\begin{equation*} S = \left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp \frac12 \\ \end{array}\right], U_1 = \left[\begin{array}{rrr} 1 \amp 0 \amp -1 \\ 0 \amp 1 \amp -1 \\ 0 \amp 0 \amp 1 \\ \end{array}\right], U_2 = \left[\begin{array}{rrr} 1 \amp -1 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right]\text{.} \end{equation*}

This gives \(U_2U_1SA = I\) so that \(A^{-1} = U_2U_1S = \left[\begin{array}{rrr} 1 \amp -1 \amp 0 \\ 0 \amp 1 \amp -\frac12 \\ 0 \amp 0 \amp \frac12 \\ \end{array}\right] \text{.}\)
