Solution 2.4.5.8.1.

  1. Since \(B\neq0\text{,}\) there is a nonzero vector \(\xvec\) such that \(B\xvec\neq\zerovec\text{.}\) However, we must have \(AB\xvec=\zerovec\text{,}\) which means that \(A(B\xvec) = \zerovec\text{.}\) There is therefore a nontrivial solution to the homogeneous equation \(A\yvec=\zerovec\) so the columns of \(A\) are linearly dependent.

  2. Since the columns of \(A\) are linearly independent, the only solution to the homogeneous equation \(A\yvec=\zerovec\) is \(\yvec=\zerovec\text{.}\) We know that \(A(B-C)\xvec = A[(B-C)\xvec]=\zerovec\) for all vectors \(\xvec\text{.}\) This means that \((B-C)\xvec=\zerovec\) or \(B\xvec=C\xvec\) for all vectors \(\xvec\text{.}\) This implies that \(B=C\text{.}\)
