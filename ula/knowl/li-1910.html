Item 3.3.4.1.b.

We will now form the vector \(\yvec\text{,}\) which is an approximation of \(\xvec\text{.}\) To do this, round all the Fourier coefficients of \(\xvec\) to the nearest integer to obtain \(\coords{\yvec}{\ccal}\text{.}\) If a coefficient has an absolute value less than one, set it equal to zero. Now find the vector \(\yvec\) and compare this approximation to \(\xvec\text{.}\) What is the error in this approximation?
