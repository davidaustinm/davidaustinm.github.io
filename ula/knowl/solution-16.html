We can use Sage to find the reduced row echelon form of the corresponding augmented matrix:

\begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 2 \amp -1 \amp 1 \\ 3 \amp 2 \amp 2 \amp 7 \\ -2 \amp 0 \amp 4 \amp -3 \\ \end{array}\right] \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 3 \\ 0 \amp 1 \amp 0 \amp -1 \\ 0 \amp 0 \amp 1 \amp 0 \end{array}\right]\text{.} \end{equation*}

This shows that there is exactly one solution \((3,-1,0)\text{.}\)
