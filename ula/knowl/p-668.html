  1. The intersection of two planes are described by a linear system having two equations and three variables. Two possible reduced row echelon forms are

    \begin{equation*} \begin{array}{cc} \left[ \begin{array}{rrr|r} 1 \amp 0 \amp * \amp * \\ 0 \amp 1 \amp * \amp * \\ \end{array} \right] \amp \left[ \begin{array}{rrr|r} 1 \amp * \amp * \amp * \\ 0 \amp 0 \amp 0 \amp 1 \\ \end{array} \right] \end{array} \end{equation*}

    indicating that the planes either intersect in a line or do not intersect at all. In particular, it is not possible for there to be a single point of intersection.

  2. If there are more variables than equations, there must be a column in the coefficient matrix that does not contain a pivot position:

    \begin{equation*} \left[ \begin{array}{rrr|r} 1 \amp 0 \amp * \amp * \\ 0 \amp 1 \amp * \amp * \\ \end{array} \right] \end{equation*}

    so we can guarantee that there are infinitely many solutions.

  3. If there are many more equations than variables, the coefficient matrix will have many rows without a pivot position:

    \begin{equation*} \left[ \begin{array}{rrr|r} 1 \amp 0 \amp * \\ 0 \amp 1 \amp * \\ 0 \amp 0 \amp * \\ 0 \amp 0 \amp * \\ 0 \amp 0 \amp * \\ 0 \amp 0 \amp * \\ 0 \amp 0 \amp * \\ \end{array} \right] \end{equation*}

    This makes it likely that there will be a pivot position in the rightmost column of the augmented matrix, which means that the linear system is inconsistent.
