Solution 6.3.4.2.1.

  1. Check that all three dot products \(\wvec_1\cdot\wvec_1=0\text{,}\) \(\wvec_1\cdot\wvec_3=0\text{,}\) and \(\wvec_2\cdot\wvec_3=0\text{.}\)

  2. \(A^TA = \begin{bmatrix} 3 \amp 0 \amp 0 \\ 0 \amp 2 \amp 0 \\ 0 \amp 0 \amp 6 \end{bmatrix} = \begin{bmatrix} \wvec_1\cdot\wvec_1\amp \wvec_1\cdot\wvec_2\amp \wvec_1\cdot\wvec_3 \\ \wvec_2\cdot\wvec_1\amp \wvec_2\cdot\wvec_2\amp \wvec_2\cdot\wvec_3 \\ \wvec_3\cdot\wvec_1\amp \wvec_3\cdot\wvec_2\amp \wvec_3\cdot\wvec_3 \\ \end{bmatrix} \text{.}\) This matrix is diagonal because the vectors form an orthogonal set.

  3. We may find the weights of this linear combination by finding \(c_i=\frac{b\cdot\wvec_i}{\wvec_i\cdot\wvec_i}\) so that \(\bvec=-2\wvec_1+3\wvec_2+2\wvec_3\text{.}\)

  4. \begin{equation*} \uvec_1=\threevec{1/\sqrt{3}}{1/\sqrt{3}} {1/\sqrt{3}},\hspace{24pt} \uvec_2=\threevec{-1/\sqrt{2}}0{1/\sqrt{2}}, \hspace{24pt} \uvec_3=\threevec{1/\sqrt{6}}{-2/\sqrt{6}} {1/\sqrt{6}},\hspace{24pt} \end{equation*}

  5. The columns of \(Q\) form an orthonormal basis for \(\real^3\) so \(Q\) is orthogonal. Therefore, \(QQ^T=I\text{.}\)
