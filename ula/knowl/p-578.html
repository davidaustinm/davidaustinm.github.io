Consider the augmented matrix

\begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 3 \\ 0 \amp 1 \amp 0 \amp -2 \\ 0 \amp 0 \amp 1 \amp 1 \\ 0 \amp 0 \amp 0 \amp 0 \\ \end{array}\right]\text{.} \end{equation*}

We know that every column of the coefficient matrix has a pivot position and that the rightmost column does not. There must be at least as many equations as unknowns.
