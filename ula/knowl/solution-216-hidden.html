1. We find that \(\vvec_1=\twovec{1}{-2}\) is an eigenvector with associated eigenvalue \(\lambda_1=2\) as is \(\vvec_2=\twovec{2}{1}\) with associated eigenvalue \(\lambda_2=-3\text{.}\) Therefore, we have \(A=PDP^{-1}\) where

    \begin{equation*} P = \mattwo12{-1}1, D=\mattwo200{-3}\text{.} \end{equation*}

  2. We see that \(\lambda=-2\) is an eigenvalue having multiplicity two. However, \(\dim E_{-2} = 1\) so we cannot find a basis for \(\real^2\) consisting of eigenvectors of \(A\text{.}\) Therefore, \(A\) is not diagonalizable.

  3. We see that \(A\) has complex eigenvalues \(\lambda=1\pm 2i\) so it is not diagonalizable in the form \(A=PDP^{-1}\) for a diagonal matrix \(D\text{.}\)

  4. The matrix \(A\) has three distinct eigenvalues \(\lambda_1=1\text{,}\) \(\lambda_2=-2\text{,}\) and \(\lambda_3=4\) with associated eigenvectors \(\vvec_1=\threevec96{-2}\text{,}\) \(\vvec_2=\threevec06{-1}\text{,}\) and \(\vvec_3=\threevec001\text{.}\) This shows us that \(A=PDP^{-1}\) where

    \begin{equation*} P = \left[\begin{array}{rrr} 9 \amp 0 \amp 0 \\ 6 \amp 6 \amp 0 \\ -2 \amp -1 \amp 1 \\ \end{array}\right], D = \left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp -2 \amp 0 \\ 0 \amp 0 \amp 4 \\ \end{array}\right]\text{.} \end{equation*}

  5. We have an eigenvalue \(\lambda=-1\) having multiplicity two, but its eigenspace has dimension two so \(A\) is diagonalizable. In particular, we choose

    \begin{equation*} P = \left[\begin{array}{rrr} 1 \amp -1 \amp 0 \\ 1 \amp 0 \amp -1 \\ 1 \amp 1 \amp 1 \\ \end{array}\right], D = \left[\begin{array}{rrr} 5 \amp 0 \amp 0 \\ 0 \amp -1 \amp 0 \\ 0 \amp 0 \amp -1 \\ \end{array}\right] \end{equation*}

    and have \(A=PDP^{-1}\text{.}\)
