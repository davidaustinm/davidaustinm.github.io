Proposition 2.6.3.

The function \(T:\real^n\to\real^m\) is a matrix transformation where \(T(\xvec) = A\xvec\) for some \(m\times n\) matrix \(A\) if and only if

\begin{equation*} \begin{aligned} T(c\vvec) \amp {}={} cT(\vvec) \\ T(\vvec + \wvec) \amp {}={} T(\vvec) + T(\wvec)\text{.} \end{aligned}\text{.} \end{equation*}

In this case, \(A\) is the matrix whose columns are \(T(\evec_j)\text{;}\) that is,

\begin{equation*} A=\left[\begin{array}{rrrr} T(\evec_1) \amp T(\evec_2) \amp \ldots \amp T(\evec_n) \end{array}\right]\text{.} \end{equation*}
