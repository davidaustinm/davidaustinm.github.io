Item 3.1.5.11.a.

  1. Since \(AB\) is invertible, \(AB\sim I\text{,}\) which says that there is pivot position in every column. Therefore, \(\xvec=\zerovec\) is the only solution to the equation \(AB\xvec = \zerovec\text{.}\)

  2. If \(\xvec\) is a solution to the equation \(B\xvec = \zerovec\text{,}\) then \(AB\xvec = A\zerovec = \zerovec\text{,}\) which says that \(\xvec = \zerovec\text{.}\)

  3. Since the only solution to the homogeneous equation \(B\xvec = \zerovec\) is the zero solution, we know that \(B\sim I\) so \(B\) is invertible.
