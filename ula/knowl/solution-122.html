Solution 3.1.5.9.1.

  1. Since \(BA\) is defined, \(B\) must have \(n\) columns. And since \(BA\) has \(n\) rows, \(B\) must have \(n\) rows. This means that \(B\) is a square \(n\times n\) matrix.

  2. Every row of \(B\) must have a pivot position, which means that \(B\) is row equivalent to the identity matrix \(I\text{.}\)

  3. Because every row of \(B\) has a pivot position, every column of \(B\) does as well. Therefore, the homogeneous equation \(B\xvec=\zerovec\) has only the trivial solution \(\xvec=\zerovec\text{.}\)

  4. If we choose any vector \(\yvec\text{,}\) we have \(B(AB-I)\yvec = B[(AB-I)\yvec]=\zerovec\text{.}\) Therefore, \((AB-I)\yvec\) is a solution to the homogeneous equation \(B\xvec=\zerovec\text{,}\) and we can conclude that \((AB-I)\yvec = \zerovec\text{.}\) This means that \(AB\yvec - \yvec = \zerovec\) or \(AB\yvec = \yvec\) for every vector \(\yvec\text{.}\) Therefore, \(AB=I\text{,}\) which shows that \(B^{-1} = A\text{.}\)
