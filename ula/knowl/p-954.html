  1. \(A = \left[\begin{array}{rrr} 2 \amp 1 \amp -3 \\ -1 \amp 2 \amp 1 \\ 3 \amp -1 \amp 0 \\ \end{array}\right]\) and \(\bvec=\threevec{4}{3}{-4}\text{.}\)

  2. Form the augmented matrix

    \begin{equation*} \left[\begin{array}{rrrr} 3 \amp -1 \amp 0 \amp -6 \\ -2 \amp 0 \amp 6 \amp 2 \end{array}\right] \sim \left[\begin{array}{rrrr} 1 \amp 0 \amp -3 \amp -1 \\ 0 \amp 1 \amp -9 \amp 3 \end{array}\right] \end{equation*}

    so that

    \begin{equation*} \xvec = \threevec{x_1}{x_2}{x_3} = \threevec{-1+3x_3}{3+9x_3}{x_3} = \threevec{-1}{3}{0}+x_3\threevec{3}{9}{1}\text{.} \end{equation*}

  3. We have the augmented matrix

    \begin{equation*} \left[\begin{array}{rrrrr} 1 \amp 2 \amp 0 \amp -1 \amp -1 \\ 2 \amp 4 \amp -3 \amp -2 \amp 1 \\ -1 \amp -2 \amp 6 \amp 1 \amp 5 \end{array}\right] \sim \left[\begin{array}{rrrrr} 1 \amp 2 \amp 0 \amp -1 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \end{array}\right]\text{.} \end{equation*}

    Since this system is inconsistent, there are no solutions to the matrix equation.

  4. We know that there is at least one solution, namely, \(\xvec=\zerovec\text{.}\)
