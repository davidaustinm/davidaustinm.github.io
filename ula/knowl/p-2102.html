  1. \(T(\xvec) = \twovec{2}{-4}\text{.}\)

  2. \(T(\xvec) = \twovec{x}{-y}\text{.}\)

  3. \(T\left(\twovec{1}{0}\right) = \twovec{1}{0}\) and \(T\left(\twovec{0}{1}\right) = \twovec{0}{-1}\text{.}\)

  4. We have the matrix \(A=\left[\begin{array}{rr} 1 \amp 0 \\ 0 \amp -1 \\ \end{array}\right]\text{.}\) It follows that \(T\left(\twovec{x}{y}\right) = A\twovec{x}{y} = \twovec{x}{-y}\) as expected.

  5. If we reflect a vector twice in the horizontal axis, we obtain the original vector. The matrix for the transformation \(T\circ T\) is simply \(AA= A^2 = \left[\begin{array}{rr}1\amp 0 \\ 0 \amp 1 \\ \end{array}\right]\text{.}\)
