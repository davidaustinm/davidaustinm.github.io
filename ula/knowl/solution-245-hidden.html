1. Remember that multiplying a matrix \(B\) by \(S\) gives a \(1\times n\) matrix whose entries are the sum of the columns of \(B\text{.}\) Therefore, \(S(A-I)=SA-SI = S-S = \left[\begin{array}{rrrr}0\amp0\amp\ldots\amp 0 \end{array}\right]\text{.}\)

  2. Any vector in \(\col(A-I)\) can be written as \((A-I)\xvec\text{.}\) However, we have \(S[(A-I)\xvec] = [S(A-I)]\xvec = [0]\text{,}\) which means that \((A-I)\xvec\) is in \(\nul(S)\text{.}\)

  3. Since the components of \(\evec_1\) add to \(1\text{,}\) \(S\evec_1=[1]\text{.}\)

  4. From the previous part, we see that \(\evec_1\) is not in \(\nul(S)\) which says it is not in \(\col(A-I)\text{.}\)

  5. If \(A-I\) is invertible, then \(\col(A-I)=\real^n\text{.}\) However, we have seen that \(\evec_1\) is not in \(\col(A-I)\) so \(A-I\) cannot be invertible.
