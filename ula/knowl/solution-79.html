Solution 2.4.5.5.1.

  1. The vector \(\xvec=\fourvec{2}{-1}{3}{1}\) is a solution.

  2. There is a nontrivial solution to the homogeneous equation \(A\xvec=\zerovec\text{.}\) Since there is also the trivial solution \(\xvec=\zerovec\text{,}\) there must be infinitely many solutions. This can only happen when there is a column of \(A\) that does not have a pivot position.

  3. \(\vvec_4 = -2\vvec_1 + \vvec_2 - 3\vvec_3\text{.}\)

  4. Because one of the vectors can be written as a linear combination of the others, the set of vectors is linearly dependent.
