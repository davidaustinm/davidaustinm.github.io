If we want to solve the equation \(A\xvec = LU\xvec = \bvec\text{,}\) we first solve for \(U\xvec = \cvec\) by solving \(L\cvec = \bvec\text{.}\) Once we have \(\cvec\text{,}\) we solve \(U\xvec = \cvec\) for the vector \(\xvec\text{,}\) which is the solution to the original equation \(A\xvec = \bvec\text{.}\) In our example,

\begin{equation*} \begin{aligned}\ L\cvec = \threevec{4}{-7}{12},\amp \qquad \cvec = \threevec{4}{1}{-1} \\ \\ U\xvec = \cvec=\threevec{4}{1}{-1}, \amp\qquad \xvec = \threevec{3}{1}{-1}\text{.} \\ \end{aligned} \end{equation*}

If we want to solve \(A\xvec = \bvec\) for a different right-hand side \(\bvec\text{,}\) we can simply repeat this two-step process.
