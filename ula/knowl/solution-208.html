Solution 4.3.5.3.1.

  1. False. A matrix can be invertible without us being able to form a basis consisting of eigenvectors. An example is \(\mattwo2102\text{.}\)

  2. True. If \(A\) and \(B\) are similar, then \(A=PBP^{-1}\) so that \(\det A = \det B\text{.}\) If \(A\) is invertible, we know that \(\det A \neq 0\text{,}\) which also tells us that \(\det B\neq 0\) and hence that \(B\) is invertible.

  3. True. If \(A=PDP^{-1}\text{,}\) then the columns of \(P\) are eigenvectors of \(A\) that form a basis for \(\real^n\text{.}\)

  4. True. If \(A=PDP^{-1}\text{,}\) then \(A^{10}=PD^{10}P^{-1}\text{.}\)

  5. False. It is possible that \(A\) has eigenvalue \(\lambda=0\text{,}\) which would imply that \(A\) is not invertible.
