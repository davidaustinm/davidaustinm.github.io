  1. After creating a Markov chain to find the steady-state vector, we find

    \begin{equation*} \begin{array}{rrrr} x_1 = 0.057, \amp x_2=0.085, \amp x_3=0.028, \amp x_4=0.085, \\ x_5 = 0.071, \amp x_6=0.212, \amp x_7=0.170, \amp x_8=0.292\text{.} \end{array} \end{equation*}

  2. This shows us that page \(8\) is judged to be the most important and page \(3\) the least important. The pages \(6\text{,}\) \(7\text{,}\) and \(8\text{,}\) are the most important. This makes sense because there are a lot of links between these pages without many links going out from these pages to the others.

  3. We have the matrix

    \begin{equation*} G = \left[\begin{array}{rrrrr} 0 \amp 0 \amp 0 \amp 0 \amp 1 \\ 1 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \amp 0 \\ \end{array}\right]\text{,} \end{equation*}

    which leads to the Markov chain

    \begin{equation*} \xvec_0=\left[\begin{array}{c} 1\\ 0\\ 0 \\ 0 \\0\end{array}\right], \xvec_1=\left[\begin{array}{c} 0\\ 1\\ 0 \\ 0 \\0\end{array}\right], \xvec_2=\left[\begin{array}{c} 0\\ 0\\ 1 \\ 0 \\0\end{array}\right], \xvec_3=\left[\begin{array}{c} 0\\ 0\\ 0 \\ 1 \\0\end{array}\right], \xvec_4=\left[\begin{array}{c} 0\\ 0\\ 0 \\ 0 \\1\end{array}\right], \ldots \end{equation*}

    This Markov chain does not converge to a steady-state vector.

  4. As all of the pages seem equally important, we should expect the PageRank vector to be

    \begin{equation*} \xvec=\left[\begin{array}{c} \frac 15 \\ \frac 15 \\ \frac 15 \\ \frac 15 \\ \frac 15 \end{array}\right]\text{.} \end{equation*}

  5. After generating some terms of a Markov chain, we see that

    \begin{equation*} \begin{array}{rrrr} x_1 = 0.000, \amp x_2=0.000, \amp x_3=0.000, \amp x_4=0.000, \\ x_5 = 0.120, \amp x_6=0.240, \amp x_7=0.240, \amp x_8=0.400\text{.} \end{array} \end{equation*}

    These are not the components of a steady-state vector because some of the entries are zero. The Google matrix \(G\) cannot be a positive matrix in this example. This is not desirable because we lose any information about the importance of the first four pages. All the PageRank drains out of the left side into the pages on the right side.
