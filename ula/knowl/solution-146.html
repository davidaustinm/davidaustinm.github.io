Solution 3.4.3.1.

  1. The matrix \(U_1\) is invertible because we see there is a pivot position in every row and column. The matrix \(U_2\text{,}\) however, is not invertible because there is not a pivot position in the third row.

  2. The determinant of an upper triangular matrix equals the product of its diagonal entries. Consequently, if the determinant of an upper triangular matrix is not zero, then each of its diagonal entries must be nonzero. In this case, there is a pivot position in every row and every column so that the matrix is invertible.

  3. Row replacement operations do not change the determinant so \(\det A = \det A_1\text{.}\)

  4. In the same way, \(\det A = \det A_2\text{.}\)

  5. Interchanges change the sign of the determinant so \(\det A = -\det U\text{.}\)

  6. The determinant \(\det U = -4\) since it is the product of the diagonal entries of \(U\text{.}\) This means that \(\det A = 4\text{.}\) We see that \(A\) is invertible because \(U\text{,}\) which has a pivot position in every row and every column, is invertible.

  7. Beginning with a row replacement operation, we arrive at

    \begin{equation*} A_1=\left[\begin{array}{rrr} 1 \amp - 1 \amp 3 \\ 0 \amp 2 \amp -2 \\ 0 \amp -3 \amp 3 \\ \end{array}\right]\text{.} \end{equation*}

    We next scale the second row by \(\frac12\) to obtain

    \begin{equation*} A_2=\left[\begin{array}{rrr} 1 \amp - 1 \amp 3 \\ 0 \amp 1 \amp -1 \\ 0 \amp -3 \amp 3 \\ \end{array}\right]\text{.} \end{equation*}

    Another row replacement operation gives

    \begin{equation*} A_3=\left[\begin{array}{rrr} 1 \amp - 1 \amp 3 \\ 0 \amp 1 \amp -1 \\ 0 \amp 0 \amp 0 \\ \end{array}\right]\text{.} \end{equation*}

    Putting these operations together, we see that \(\det A = 2\det U = 0\text{.}\) In this case, \(A\) is not invertible because \(U\text{,}\) which has a row without a pivot position, is not invertible.

  8. Performing one of the three row operations either leaves the determinant unchanged (row replacement), changes its sign (interchange), or multiplies it by a nonzero number (scaling). Therefore, if we begin with a matrix whose determinant is not zero, the determinant remains not zero after any row operation is applied.

  9. If we apply a sequence of row operations to \(A\) to find a row equivalent matrix \(U\) that is upper triangular, we know that \(\det A\neq0\) if and only if \(\det U \neq 0\text{.}\) We also know that \(A\) is invertible if and only if \(U\) is invertible. Putting these facts together, we conclude that \(\det A\neq 0\) if and only if \(A\) is invertible.

  10. Since \(A^{-1}A = I\text{,}\) we have \(\det A^{-1} \det A = 1\) so \(\det A^{-1} = \frac 13\text{.}\)
