1. We form the matrix

    \begin{equation*} P_{\bcal} = \left[\begin{array}{rrrrrr} 1 \amp 1 \amp 1 \amp 1 \amp 1 \amp 1 \\ 0 \amp 1 \amp 1 \amp 1 \amp 1 \amp 1 \\ 0 \amp 0 \amp 1 \amp 1 \amp 1 \amp 1 \\ 0 \amp 0 \amp 0 \amp 1 \amp 1 \amp 1 \\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \amp 1 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 1 \end{array}\right]\text{.} \end{equation*}

  2. We find that

    \begin{equation*} P_{\bcal}^{-1} = \left[\begin{array}{rrrrrr} 1 \amp -1 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp -1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp -1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \amp -1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \amp -1 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 1 \end{array}\right]\text{.} \end{equation*}

  3. We see that \(c_2=x_3-x_2\) so \(c_2\) measures the change in brightness between one pixel and its neighbor. Similarly, \(c_4=x_5-x_4\text{,}\) which measures another change in brightness.

  4. We compute that

    \begin{equation*} \coords{\xvec}{\bcal}=P_{\bcal}^{-1}\xvec = \left[\begin{array}{c} -9 \\ 4 \\ -15 \\ -145 \\ -10 \\ 200 \end{array}\right]\text{.} \end{equation*}

  5. Most of the coefficients that measure changes are relatively small in absolute value. The coefficient \(c_4\text{,}\) however, which measures the change in brightness between the fourth and fifth pixel, has a large absolute value. This tells us that there is a large change in brightness between the fourth and fifth pixel, which points to an edge in the image.
