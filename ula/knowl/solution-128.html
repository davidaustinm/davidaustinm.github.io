Solution 3.2.5.2.1.

  1. In both cases, we see that

    \begin{equation*} C_{\bcal}=\left[\begin{array}{rr} \vvec_1\amp\vvec_2 \end{array}\right] \sim \left[\begin{array}{rr} 1\amp0\\ 0\amp1\\ \end{array}\right], C_{\ccal}=\left[\begin{array}{rr} \wvec_1\amp\wvec_2 \end{array}\right] \sim \left[\begin{array}{rr} 1\amp0\\ 0\amp1\\ \end{array}\right] \end{equation*}

    so that both sets of vectors are linearly independent and span \(\real^2\text{.}\)

  2. We solve \(C_{\bcal}\coords{\xvec}{\bcal} = \xvec\) and \(C_{\ccal}\coords{\xvec}{\ccal} = \xvec\) to find \(\coords{\xvec}{\bcal}=\twovec{12/5}{13/5}\) and \(\coords{\xvec}{\ccal}=\twovec{13}{21}\text{.}\)

  3. We have \(\xvec=C_{\bcal}\coords{\xvec}{\bcal} = \twovec{-2}{16}\text{.}\) We then solve \(C_{\ccal}\coords{\xvec}{\ccal} = \twovec{-2}{16}\) to find \(\coords{\xvec}{\ccal}=\twovec{-20}{-38}\text{.}\)

  4. In the same way, we find \(\xvec=\twovec{-8}{-13}\) and \(\coords{\xvec}{\ccal}=\twovec{-37/5}{-3/5}\text{.}\)

  5. We have \(\coords{\xvec}{\bcal} = C_{\bcal}^{-1}\xvec = C_{\bcal}^{-1}C_{\ccal}\coords{\xvec}{\ccal}\text{,}\) which shows that \(D = C_{\bcal}^{-1}C_{\ccal}= \left[\begin{array}{rr} \frac95 \amp -1 \\ \frac15 \amp 0 \\ \end{array}\right] \text{.}\)
