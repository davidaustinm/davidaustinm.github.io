Solution 3.2.5.9.1.

  1. The vectors are linearly independent and span \(\real^2\text{.}\)

  2. We compute that \(A\vvec_1 = 3\vvec_1\) and \(A\vvec_2=\vvec_2\text{.}\)

  3. \(\coords{A\vvec_1}{\bcal} = \twovec{3}{0}\) and \(\coords{A\vvec_2}{\bcal} = \twovec{0}{1}\text{.}\)

  4. We know that \(\xvec = \vvec_1-5\vvec_2\text{,}\) which means that \(A\xvec = 3\vvec_1-5\vvec_2\text{.}\) Therefore, \(\coords{A\xvec}{\bcal}=\twovec{3}{-5}\text{.}\)

  5. If \(\coords{\xvec}{\bcal} = \twovec{c_1}{c_2}\text{,}\) then \(\xvec=c_1\vvec_1+c_2\vvec_2\) and \(A\xvec=3c_1\vvec_1+c_2\vvec_2\text{.}\) Therefore, \(D\twovec{c_1}{c_2} = \twovec{3c_1}{c_2}\text{,}\) which says that \(D= \left[\begin{array}{rr} 3 \amp 0 \\ 0 \amp 1 \\ \end{array}\right] \text{.}\)
