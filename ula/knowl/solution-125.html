Solution 3.1.5.5.1.

  1. If

    \begin{equation*} P = \left[\begin{array}{rrr} 0 \amp 1 \amp 0 \\ 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right], S_1 = \left[\begin{array}{rrr} -\frac13 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right], S_2 = \left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp \frac12 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right]\text{,} \end{equation*}

    then \(S_2S_1P A = I\) so that \(A^{-1}=S_2S_1P= \left[\begin{array}{rrr} 0 \amp -\frac13 \amp 0 \\ \frac12 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] \text{.}\)

  2. We use three replacement operations

    \begin{equation*} \begin{array}{l} L_1 = \left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ -2 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \\ \end{array}\right], L_2 = \left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 3 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \\ \end{array}\right], \\ L_3 = \left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp -2 \amp 1 \\ \end{array}\right]\text{.} \end{array} \end{equation*}

    Therefore, \(L_3L_2L_1A = I\) so \(A^{-1} = L_3L_2L_1 = \left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ -2 \amp 1 \amp 0 \amp 0 \\ -6 \amp 3 \amp 1 \amp 0 \\ 12 \amp -6 \amp -2 \amp 1 \\ \end{array}\right] \text{.}\)

  3. We use a scaling followed by replacement operations:

    \begin{equation*} S = \left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp \frac12 \\ \end{array}\right], U_1 = \left[\begin{array}{rrr} 1 \amp 0 \amp -1 \\ 0 \amp 1 \amp -1 \\ 0 \amp 0 \amp 1 \\ \end{array}\right], U_2 = \begin{bmatrix} 1 \amp 0 \amp -1 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{bmatrix}, U_3 = \left[\begin{array}{rrr} 1 \amp -1 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right]\text{.} \end{equation*}

    This gives \(U_3U_2U_1SA = I\) so that \(A^{-1} = U_3U_2U_1S = \left[\begin{array}{rrr} 1 \amp -1 \amp 0 \\ 0 \amp 1 \amp -\frac12 \\ 0 \amp 0 \amp \frac12 \\ \end{array}\right] \text{.}\)
