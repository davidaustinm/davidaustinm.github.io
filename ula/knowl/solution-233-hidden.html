1. If we add the entries in each column of \(A\) and each column of \(B\text{,}\) we obtain \(1\text{.}\) Also, all the entries in both matrices are nonnegative.

  2. The matrix \(A\) has the eigenvalues \(\lambda_1 = 1\) and \(\lambda_2=-1\) with associated eigenvectors \(\vvec_1=\twovec11\) and \(\vvec_2=\twovec{-1}{1}\text{.}\) The steady-state vector is \(\qvec=\twovec{\frac12}{\frac12}\) as this is the unique probability vector in \(E_1\text{.}\)

  3. The terms in the Markov chain are

    \begin{equation*} \xvec_0 = \twovec10, \xvec_1 = \twovec01, \xvec_2 = \twovec10, \xvec_3 = \twovec01,\ldots \end{equation*}

    so the chain does not converge to any vector, much less the steady-state vector.

  4. The matrix \(B\) has eigenvalues \(\lambda_1=1\) and \(\lambda_2=0.1\) with associated eigenvectors \(\vvec_1=\twovec12\) and \(\vvec_2=\twovec{-1}{1}\text{.}\) The unique steady-state vector is \(\qvec=\twovec{\frac13}{\frac23}\) since this is the only probability vector in \(E_1\text{.}\)

  5. Here we find

    \begin{equation*} \begin{array}{rrrr} \xvec_0 = \twovec10, \amp \xvec_1 = \twovec{0.4}{0.6}, \amp \xvec_2 = \twovec{0.34}{0.66}, \\ \xvec_3 = \twovec{0.334}{0.666}, \amp \xvec_4 = \twovec{0.3334}{0.6666}, \amp \xvec_5 = \twovec{0.33334}{0.66666},\amp\ldots \end{array} \end{equation*}

    which appears to be converging to the steady-state vector \(\qvec=\twovec{\frac13}{\frac23}\text{.}\)

  6. If there is one eigenvalue \(\lambda_1=1\) having multiplicity one with the other eigenvalues satisfying \(|\lambda_j|\lt 1\text{,}\) we can guarantee that any Markov chain will converge to a unique steady-state vector.
