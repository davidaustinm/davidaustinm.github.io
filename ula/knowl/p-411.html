  1. We will form a system of linear equations by considering each intersection and equating the number of cars that enter the intersection to the number of cars that leave.

    1. For the intersection in the upper left, the number of cars that enter is \(x\) while the number of cars that leave is \(y + 100\text{.}\) This gives the equation \(x = y + 100\) or \(x-y = 100\text{.}\)

    2. By studying the other three intersections, we form a linear system consisting of four equations in four unknowns:

      \begin{equation*} \begin{alignedat}{5} x \amp {}-{} \amp y \amp \amp \amp \amp \amp {}={} \amp 100 \\ \amp \amp y \amp {}-{} \amp z \amp \amp \amp {}={} \amp -250 \\ \amp \amp \amp \amp \amp \amp w \amp {}={} \amp 160 \\ \amp \amp \amp \amp z \amp {}-{} \amp w \amp {}={} \amp 390\text{.} \\ \end{alignedat} \end{equation*}

      If we express these equations in terms of an augmented matrix and use Sage to find its reduced row echelon form, we find

      \begin{equation*} \left[\begin{array}{rrrr|r} 1 \amp -1 \amp 0 \amp 0 \amp 100 \\ 0 \amp 1 \amp -1 \amp 0 \amp -250 \\ 0 \amp 0 \amp 0 \amp 1 \amp 160 \\ 0 \amp 0 \amp 1 \amp -1 \amp 390 \end{array}\right] \sim \left[\begin{array}{rrrrr} 1 \amp 0 \amp 0 \amp 0 \amp 400 \\ 0 \amp 1 \amp 0 \amp 0 \amp 300 \\ 0 \amp 0 \amp 1 \amp 0 \amp 550 \\ 0 \amp 0 \amp 0 \amp 1 \amp 160 \end{array}\right] \end{equation*}

    3. There is exactly one solution: \(x=400\text{,}\) \(y=300\text{,}\) \(z=550\text{,}\) and \(w=160\text{.}\) This is what we expect since the traffic flow \(w\) is determined by the upper right intersection. From here, all the other flows are determined as well.

  2. Let's now look at another traffic pattern.

    1. In this case, we find the equations:

      \begin{equation*} \begin{alignedat}{5} x \amp {}+{} \amp y \amp \amp \amp \amp \amp {}={} \amp 650 \\ x \amp \amp \amp {}-{} \amp z \amp \amp \amp {}={} \amp -150 \\ \amp \amp y \amp \amp \amp {}+{} \amp w \amp {}={} \amp 580 \\ \amp \amp \amp \amp z \amp {}-{} \amp w \amp {}={} \amp 220\text{.} \\ \end{alignedat} \end{equation*}

      This gives the augmented matrix:

      \begin{equation*} \left[\begin{array}{rrrr|r} 1 \amp 1 \amp 0 \amp 0 \amp 650 \\ 1 \amp 0 \amp -1 \amp 0 \amp -150 \\ 0 \amp 1 \amp 0 \amp 1 \amp 580 \\ 0 \amp 0 \amp 1 \amp -1 \amp 220 \end{array}\right] \sim \left[\begin{array}{rrrrr} 1 \amp 0 \amp 0 \amp -1 \amp 70 \\ 0 \amp 1 \amp 0 \amp 1 \amp 580 \\ 0 \amp 0 \amp 1 \amp -1 \amp 220 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \end{array}\right]\text{.} \end{equation*}

    2. Here, we may view \(w\) as a free variable and write

      \begin{equation*} \begin{alignedat}{3} x \amp {}={} \amp 70 + w \\ y \amp {}={} \amp 580 - w \\ z \amp {}={} \amp 220 + w\text{.} \\ \end{alignedat} \end{equation*}

      This shows that there are infinitely many solutions. This makes sense since the unknown flows \(x\text{,}\) \(y\text{,}\) \(z\text{,}\) and \(w\) form a loop. For instance, we can add one car to each of \(x\text{,}\) \(z\text{,}\) and \(w\) provided we remove one from \(y\text{.}\)

    3. Since we must have \(y\geq 0\text{,}\) it follows that the free variable \(w\) satisfies \(0\leq w \leq 580\text{.}\) This means that \(x\) satisfies \(70 \leq x \leq 650\text{.}\)
