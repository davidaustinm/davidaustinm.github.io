Solution 1.4.4.6.1.

  1. Since all the entries in the rightmost column are zero, there cannot be a pivot in the rightmost column.

  2. We have a solution when all the variables are zero. Therefore, the system must be consistent.

  3. We must have a pivot position in every column of the coefficient matrix so the augmented matrix could look like

    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp * \\ 0 \amp 1 \amp 0 \amp * \\ 0 \amp 0 \amp 1 \amp * \\ 0 \amp 0 \amp 0 \amp 0 \\ \end{array}\right]\text{.} \end{equation*}

    Since every column of the coefficient matrix has a pivot position, there must be at least as many rows as columns. This means that there must be at least as many equations as unknowns.
