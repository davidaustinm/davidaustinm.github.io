  1. We compute that

    \begin{equation*} \left[\begin{array}{rrr} 1 \amp 2 \amp -4 \\ -1 \amp -1 \amp 3 \\ 0 \amp -1 \amp 2 \end{array}\right] \left[\begin{array}{rrr} 1 \amp 0 \amp 2 \\ 2 \amp 2 \amp 1 \\ 1 \amp 1 \amp 1 \end{array}\right] = \left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{array}\right] \end{equation*}

    showing that \(B=A^{-1}\text{.}\)

  2. Since we have \(B=A^{-1}\text{,}\) we compute \(\xvec=B\threevec{4}{-1}{4} = \threevec{-14}{9}{9}\text{.}\)

  3. We also see that \(AB=I\text{.}\)

  4. Since the equation \(A\xvec=\bvec\) is consistent for every \(\bvec\text{,}\) the columns of \(A\) must span \(\real^n\text{.}\)

  5. This means that there is a pivot position in every row. Since the matrix has the same number of rows and columns, there is also a pivot position in every column.

  6. Since there is a pivot position in every row and every column, the reduced row echelon form of \(A\) is the identity matrix \(I\text{.}\)
