Solution 3.5.2.1.

  1. If \(\vvec\) is a vector whose tip lies on the line, then any scalar multiple of \(\vvec\) must lie on the line if the line is to be a subspace. In particular, \(0\vvec=\zerovec\) would have to lie on the line, which is not the case.

  2. If \(\vvec\) is a vector in the subspace, then \(0\vvec=\zerovec\) must be in the subspace as well.

  3. Any linear combination of \(\zerovec\) is also \(\zerovec\text{,}\) which means it is contained in the subset.

  4. Any linear combination of vectors in \(\real^2\) is also a vector in \(\real^2\text{.}\)

  5. If we remember that \(\span{\vvec,\wvec}\) is the set of linear combinations of \(\vvec\) and \(\wvec\text{,}\) it follows that any vector in \(\span{\vvec, \wvec}\) is a linear combination of \(\vvec\) and \(\wvec\) and is therefore a vector in \(S\text{.}\)

  6. The vectors span \(\real^2\) so the subspace \(S=\real^2\text{.}\)
