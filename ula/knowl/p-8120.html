  1. If \(\vvec\) is an eigenvector of \(G\) with eigenvalue \(\lambda\text{,}\) then \(\vvec\cdot(G\vvec) = \lambda\vvec\cdot\vvec= \lambda|\vvec|^2 = |A\vvec|^2\) so \(\lambda\geq0\text{.}\)

  2. If \(A\) is symmetric, then \(G=A^TA = A^2\text{.}\) Then if \(A\vvec=\lambda \vvec\text{,}\) it follows that \(G\vvec = \lambda^2\vvec\text{.}\)

  3. If \(\lambda\) is an eigenvalue of \(A\text{,}\) then \(\sigma^2 = \lambda^2\) is an eigenvalue of \(G\text{.}\) Then \(\sigma = |\lambda|\text{.}\)
