  2. The matrix is \(E=\left[\begin{array}{rr} 0 \amp -1 \\ 1 \amp 0 \\ \end{array}\right]\text{,}\) which is a \(90^\circ\) rotation.

  3. If we rotate by \(90^\circ\) four times, we obtain the identity. Therefore, \(A^4=I\text{.}\)

  4. Here, we find the eigenvalues \(\lambda=\frac12\pm \frac{\sqrt{3}}2 i\text{.}\)

  5. The matrix is \(E=\left[\begin{array}{rr} \frac 12 \amp -\frac{\sqrt{3}}2 \\ \frac{\sqrt{3}}2 \amp \frac 12\end{array}\right]\text{,}\) which is a \(60^\circ\) rotation.

  6. If we rotate by \(60^\circ\) six times, we obtain the identity. Therefore, \(B^6=I\text{.}\)
