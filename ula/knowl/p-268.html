  1. The reduced row echelon form is

    \begin{equation*} \left[\begin{array}{rr|r} 1 \amp 0 \amp -1 \\ 0 \amp 1 \amp 2 \\ 0 \amp 0 \amp 0 \\ \end{array}\right]\text{,} \end{equation*}

    and there is a single solution. Every variable is basic.

  2. The reduced row echelon form is

    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp -1 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 1 \\ \end{array}\right]\text{.} \end{equation*}

    All the variables are basic so there is a unique solution: \(x_1 = -1\text{,}\) \(x_2=0\text{,}\) and \(x_3=1\text{.}\)

  3. The reduced row echelon form is

    \begin{equation*} \left[\begin{array}{rrrr|r} 1 \amp 0 \amp -1 \amp 0 \amp 2 \\ 0 \amp 1 \amp -2 \amp 0 \amp -3 \\ 0 \amp 0 \amp 0 \amp 1 \amp -1 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \end{array}\right]\text{,} \end{equation*}

    which gives the equations

    \begin{equation*} \begin{aligned} x_1 \amp = 2 + x_3 \\ x_2 \amp = -3 + 2x_3 \\ x_4 \amp = -1 - x_3. \\ \end{aligned} \end{equation*}

    This shows that \(x_3\) is a free variable and \(x_1\text{,}\) \(x_2\text{,}\) and \(x_4\) are basic variables. There are therefore infinitely many solutions.
