Let's summarize the results of this activity by considering the following reduced row echelon matrix:

\begin{equation*} \left[ \begin{array}{rrr|r} 1 \amp * \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \amp 0 \\ \end{array} \right]\text{.} \end{equation*}

In terms of variables \(x\text{,}\) \(y\text{,}\) and \(z\text{,}\) the final equation says

\begin{equation*} 0x + 0y + 0z = 0\text{.} \end{equation*}

If we evaluate the left-hand side with any values of \(x\text{,}\) \(y\text{,}\) and \(z\text{,}\) we get 0, which means that the equation always holds. Therefore, its presence has no effect on the solution space defined by the other three equations.
