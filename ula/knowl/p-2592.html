Looking at the reduced row echelon form, we find

\begin{equation*} \left[\begin{array}{rrrrr} \vvec_1 \amp \vvec_2 \amp \vvec_3 \amp \vvec_4 \amp \vvec_5 \\ \end{array}\right] \sim \left[\begin{array}{rrrrr} 1 \amp 0 \amp - 2 \amp 0 \amp -1 \\ 0 \amp 1 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \amp -1 \\ \end{array}\right]\text{.} \end{equation*}

This shows that the vectors are not linearly independent since there is not a pivot position in every column. Therefore, the set of vectors does not form a basis for \(\real^3\text{.}\) Of course, we also know this because a set of vectors for \(\real^3\) must contain exactly three vectors.
