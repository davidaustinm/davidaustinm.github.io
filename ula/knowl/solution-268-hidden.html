1. We begin with \(E_1A=A_1\) where

    \begin{equation*} E_1=\left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ -1 \amp 1 \amp 0 \amp 0 \\ -1 \amp 0 \amp 1 \amp 0 \\ -1 \amp 0 \amp 0 \amp 1 \\ \end{array}\right], A_1= \left[\begin{array}{rrrr} a \amp a \amp a \amp a \\ 0 \amp b-a \amp b-a \amp b-a \\ 0 \amp b-a \amp c-a \amp c-a \\ 0 \amp b-a \amp c-a \amp d-a \\ \end{array}\right]\text{.} \end{equation*}

    Next we have \(E_2A_1=A_2\) where

    \begin{equation*} E_2=\left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp -1 \amp 1 \amp 0 \\ 0 \amp -1 \amp 0 \amp 1 \\ \end{array}\right], A_2= \left[\begin{array}{rrrr} a \amp a \amp a \amp a \\ 0 \amp b-a \amp b-a \amp b-a \\ 0 \amp 0 \amp c-b \amp c-b \\ 0 \amp 0 \amp c-b \amp d-b \\ \end{array}\right]\text{.} \end{equation*}

    Finally, we have \(E_3A_2=U\text{,}\) an upper triangular matrix where

    \begin{equation*} E_3=\left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp -1 \amp 1 \\ \end{array}\right], U = \left[\begin{array}{rrrr} a \amp a \amp a \amp a \\ 0 \amp b-a \amp b-a \amp b-a \\ 0 \amp 0 \amp c-b \amp c-b \\ 0 \amp 0 \amp 0 \amp d-c \\ \end{array}\right]\text{.} \end{equation*}

    This gives

    \begin{equation*} L = \left[\begin{array}{rrrr} 1 \amp 0 \amp 0 \amp 0 \\ 1 \amp 1 \amp 0 \amp 0 \\ 1 \amp 1 \amp 1 \amp 0 \\ 1 \amp 1 \amp 1 \amp 1 \\ \end{array}\right], U = \left[\begin{array}{rrrr} a \amp a \amp a \amp a \\ 0 \amp b-a \amp b-a \amp b-a \\ 0 \amp 0 \amp c-b \amp c-b \\ 0 \amp 0 \amp 0 \amp d-c \\ \end{array}\right]\text{.} \end{equation*}

  2. Notice that \(A\) is invertible if and only if \(U\) is invertible and \(U\) is invertible if and only all the diagonal terms are not equal. This means that \(A\) is invertible provided that \(a\neq0\text{,}\) \(b\neq a\text{,}\) \(c\neq b\text{,}\) and \(d\neq c\text{.}\)
