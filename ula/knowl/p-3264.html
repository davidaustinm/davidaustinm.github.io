This expression says that any vector \(\xvec\) satisfying \(A\xvec= \zerovec\) is a linear combination of the vectors

\begin{equation*} \vvec_1 = \fivevec{2}{-1}{1}{0}{0}, \vvec_2 = \fivevec{3}{-1}{0}{1}{0}, \vvec_3 = \fivevec{0}{2}{0}{0}{1}\text{.} \end{equation*}

It is easy to see that these vectors are linearly independent. Remember that we saw in SectionÂ 2.4 that this set of vectors is linearly dependent if any linear combination \(c_1\vvec_1 + c_2\vvec_2 + c_3\vvec_3 = \zerovec\) implies that \(c_1=c_2=c_3 = 0\text{.}\) But this linear combination would be

\begin{equation*} \begin{aligned} c_1\vvec_1 + c_2\vvec_2 + c_3\vvec_3 \amp {}={} c_1\fivevec{2}{-1}{1}{0}{0} +c_2\fivevec{3}{-1}{0}{1}{0} +c_3\fivevec{0}{2}{0}{0}{1} \\ \\ \amp {}={} \fivevec{2c_1 + 3c_2}{-c_1-c_2+2c_3}{c_1}{c_2}{c_3} = \fivevec{0}{0}{0}{0}{0} \end{aligned}\text{.} \end{equation*}

This expression shows that \(c_1=c_2=c_3=0\) so the vectors are linearly independent.
